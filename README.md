A modern and responsive landing page built for **CyberClasses Sirsa**, showcasing courses and tech learning opportunities.

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Animations  
- **Spline** – 3D visuals integration  

---

## ✨ Features

- 🔥 Clean and modern UI design  
- 📱 Responsive (Mobile + Desktop)  
- 🎨 Gradient effects & animations  
- 🤖 3D robot integration using Spline  
- ⚡ Fast and lightweight
